## Projects

### 1. [Secure Multi-tier Web App on AWS](./secure-multi-tier-webapp)
- **Description:** Multi-tier web application with secure AWS architecture
- **Frontend:** React hosted on S3 + CloudFront
- **Backend:** Python/Node.js API on EC2/Lambda
- **Database:** RDS in private subnet
- **Automation:** Terraform for infrastructure, GitHub Actions for CI/CD
- **Security:** IAM roles, Security Groups, AWS WAF
- **Monitoring:** CloudWatch for logs and metrics
- [Project README & Code](./secure-multi-tier-webapp)
